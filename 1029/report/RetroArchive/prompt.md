“레트로 아카이브(Retro Archive)”라는 웹사이트를 만들어줘. 이 사이트는 시대와 국가를 비교하며 문화를 탐색하는 인터랙티브 아카이브야. 사용자는 연도(1960~2000년대)와 국가(한국, 미국, 일본 등)를 선택해 음악·광고·오락, 패션, 자동차 등의 시대별 문화를 비교할 수 있어.

📄 페이지 구성: 홈: 잔잔하고 따뜻한 복고풍 분위기의 메인 화면. 배경은 오래된 오락기나 브라운관 느낌, 살짝 바랜 색감(베이지·브라운·파스텔 톤). 제목 “Retro Archive”, 서브문구 “시대와 국가를 넘나드는 문화 비교 플랫폼”. 중앙에 옛날 게임기 화면이 있고, “시작하기” 버튼을 누르면 게임 안으로 들어가듯 비교 페이지로 이동.

비교 뷰어 페이지: 국가/연도 선택 드롭다운과 슬라이더. 선택 시 세 국가의 시대별 카드 3개가 나란히 표시되어 음악, 광고, 게임, 패션, 자동차 키워드 비교. 각 카드 클릭 시 상세 페이지로 이동.

시대 상세 페이지: 인터랙티브 타임라인(연도별 사건, 유행, 인물), 이미지/영상 갤러리(광고, 잡지, 게임 화면), 문화 키워드 태그(‘워크맨’, ‘청바지’ 등), “더 알아보기” 버튼으로 해설 페이지 이동.

커뮤니티 페이지: “내가 겪은 90년대” 같은 회고글 업로드 섹션. 카드형 게시물, 좋아요 버튼, 댓글 가능.

색상: 연한 회색, 베이지, 크림, 브라운, 파스텔 민트 정도

픽셀아트나 도트그래픽 요소 약간 사용

전체적으로 조용하고 감성적인 ‘아날로그 레트로’ 느낌

💡 기능 요약: 국가·연도 선택, 타임라인, 이미지/영상 아카이브, 문화 키워드 검색, 사용자 회고 업로드.

전체적으로 ‘세대 간 소통과 문화 비교’를 부드럽고 향수를 자극하는 방식으로 보여주는 구조로 만들어줘.
