## ✅ 1. 최종 버전 (따뜻한 브라운관 느낌)

### 🎯 프로젝트 개요
> “레트로 아카이브(Retro Archive)”는 시대와 국가를 비교하며 문화를 탐색하는 인터랙티브 아카이브입니다.  
> 사용자는 연도(1960~2000년대)와 국가(한국, 미국, 일본 등)를 선택해  
> **음악 · 광고 · 오락 · 패션 · 자동차** 등의 문화를 한눈에 비교할 수 있습니다.

---

## 📄 페이지 구성

### 🏠 홈 (Home)
- **분위기:** 잔잔하고 따뜻한 복고풍  
- **배경:** 오래된 오락기, 브라운관 질감, 살짝 바랜 색감  
- **타이틀:**  
  - 제목: `Retro Archive`  
  - 서브문구: `시대와 국가를 넘나드는 문화 비교 플랫폼`
- 중앙에 옛날 게임기 화면이 있고,  
  `시작하기` 버튼 클릭 시 비교 페이지로 이동 (게임 안으로 들어가는 연출)

---

### 🔍 비교 뷰어 페이지 (Comparison Viewer)
- **UI 요소:**  
  - 국가 선택 드롭다운  
  - 연도 슬라이더  
- **기능:**  
  선택된 조건에 따라 세 국가의 시대별 카드 3개가 나란히 표시됨  
  (예: 한국 / 미국 / 일본의 음악, 광고, 게임, 패션, 자동차)
- 각 카드 클릭 시 → **시대 상세 페이지로 이동**

---

### 🕰️ 시대 상세 페이지 (Era Detail)
- **구성 요소:**  
  - 인터랙티브 타임라인 (연도별 사건, 유행, 인물)  
  - 이미지·영상 갤러리 (광고, 잡지, 게임 화면 등)  
  - 문화 키워드 태그 (`워크맨`, `청바지`, `브라운관` 등)  
  - “더 알아보기” 버튼 → 해설 페이지 이동

---

### 💬 커뮤니티 페이지 (Community)
- 주제 예시: “내가 겪은 90년대”  
- **기능:**  
  - 회고글 업로드  
  - 카드형 게시물  
  - 좋아요 버튼, 댓글 기능  

---
